Betterer Configuration

Betterer options#

Whenever you run Betterer (from the CLI or via the JS API) you can pass an options object. It will be validated by Betterer and turned into a BettererConfig.

Base options#

type BettererOptionsBase = Partial<{  configPaths: Array<string> | string;  cwd: string;  filters: Array<string | RegExp> | string;  reporters: Array<string | BettererReporter>;  resultsPath: string;  silent: boolean;  tsconfigPath: string;}>;

configPaths: Array<string> | string | string (default: ['./.betterer.ts])#

The list of test definition file paths containing Betterer tests for the current run.

cwd: string (default: process.cwd())#

The current working directory for the current run.

filters: Array<string | RegExp> | string (default: [])#

The list of filters to select tests for the current run. Will be parsed into BettererConfigFilters.

reporters: BettererConfigReporters (default: ['@betterer/reporter'])#

The list of reporters for the current run.

resultsPath: string (default: '.betterer.results')#

The path to the results file for the current run.

silent: boolean (default: true)#

When set to true, all default reporters will be disabled. Custom reporters will still work normally.

tsconfigPath: string | null (default: null)#

The path to the TypeScript configuration for the current run.

BettererOptionsRunner#

Options object for creating a Betterer runner. It will be validated by Betterer and turned into a BettererConfig.

type BettererOptionsRunner = BettererOptionsBase &  Partial<{    ignores: BettererConfigIgnores;  }>;

All the base options above as well as:

ignores: BettererConfigIgnores (default: [])#

The list of glob patterns to ignore from file watching for the current run.

BettererOptionsStart#

Options object for a Betterer run in default mode. It will be validated by Betterer and turned into a BettererConfig.

type BettererOptionsStart = BettererOptionsBase &  Partial<{    ci: boolean;    excludes: Array<string | RegExp> | string;    includes: Array<string> | string;    strict: boolean;    update: boolean;  }>;

All the base options above as well as:

ci: boolean (default: false)#

When set to true, CI mode is enabled. In [CI mode] Betterer will throw an error if there is any difference between the test results and the expected results.

excludes: Array<string | RegExp> | string (default: [])#

The list of filters to exclude files for the current run. Each file path declared by the includes patterns will be matched against these patterns.

includes: Array<string> | string (default: [])#

The list of globs to select files for the current run. The expanded list of file paths will be fiiltered by excludes.

strict: boolean (default: false)#

When set to true, Strict mode is enabled. In [strict mode] Betterer will throw an error if there is any difference between the test results and the expected results. Set to true when ci is true.

update: boolean (default: false)#

When set to true, Betterer will update the results file, even if things get worse. Set to false when ci or strict are true.

BettererOptionsWatch#

Options object for a Betterer run in Watch mode. Will be parsed into a BettererConfig.

export type BettererOptionsWatch = BettererOptionsRunner &  Partial<{    watch: true;  }>;

All the base options above as well as:

ignores: BettererConfigIgnores (default: [])#

The list of glob patterns to ignore from file watching for the current run.

watch: true#

Must be set to true in Watch mode. In [Watch mode] Betterer will run your tests any time a file changes.

BettererConfig#

A validated configuration object for a Betterer run:

type BettererConfig = {  ci: boolean;  configPaths: BettererConfigPaths;  cwd: string;  filePaths: BettererConfigPaths;  filters: BettererConfigFilters;  ignores: BettererConfigIgnores;  reporters: BettererConfigReporters;  resultsPath: string;  silent: boolean;  strict: boolean;  tsconfigPath: string | null;  update: boolean;  watch: boolean;};

BettererConfigFilters#

A list of regular expressions to filter which tests to run.

type BettererConfigFilters = ReadonlyArray<RegExp>;

BettererConfigIgnores#

A list of glob patterns to ignore from file watching.

type BettererConfigIgnores = ReadonlyArray<string>;

BettererConfigPaths#

A list of test definition file paths containing Betterer tests.

type BettererConfigPaths = ReadonlyArray<string>;

BettererConfigReporter#

A path to a module that exports a BettererReporter or an inline BettererReporter.

type BettererConfigReporter = string | BettererReporter;

BettererConfigReporters#

A list of BettererConfigReporter.

type BettererConfigReporter = ReadonlyArray<BettererConfigReporter>;
